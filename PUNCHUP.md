# PORTFOLIO PUNCH UP PROPOSAL
# Idea
My revised idea for the portfolio punch up is an dynamic custom cursor.
    - Will be coded using static HTML, CSS, and JavaScript.
    - HTML may be injected into the JavaScript file (JavaScript Injection).
    - Will be a star with eyes and a mouth made using CSS.
    - The cursor will rotate slowly with CSS animations while user is browsing.
    - When hovering over a button or card that leads to content cursor will wiggle
    and a speech bubble will appear with either "View [Content]?" if the interactive
    element leads to a page or "Coming Soon!" if the page is under construction.
    - The face will also change when "Coming Soon!" is said.
    - When hovering, the cursor will also scale in size.
    - GSAP may also be used to make the animations more polished such as text animation
    for the speech bubble and mouth movement when speech bubble pops up if possible.
